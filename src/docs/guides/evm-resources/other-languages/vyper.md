# Vyper

[Vyper](https://www.quicknode.com/guides/ethereum-development/smart-contracts/how-to-write-an-ethereum-smart-contract-using-vyper) 是 EVM 的一种流行编程语言，在逻辑上类似于 Solidity，在语法上类似于 Python。

[Vyper 文档](https://docs.vyperlang.org/en/stable/index.html) 涵盖了安装 Vyper 语言、语言语法、编码示例、编译等内容。

建议寻求类 Python 体验的典型 EVM 开发者使用 Vyper 作为编程语言，使用 [ApeWorx](https://docs.apeworx.io/ape/stable/userguides/quickstart.html) 作为测试和部署框架，它利用 Python 语言。ApeWorx 还允许在测试结果分析中使用典型的 Python 库，如 Pandas。

Vyper 和 ApeWorx 可以与 [Jupyter](https://jupyter.org/) 一起使用，它提供了使用网络浏览器的交互式环境。关于为 EVM 智能合约开发使用 Vyper 和 Jupyter 的快速设置指南可以在[这里](https://medium.com/deepyr/interacting-with-ethereum-using-web3-py-and-jupyter-notebooks-e4207afa0085)找到。
