### 引言

Monad 是一个高性能的以太坊兼容一级区块链（L1）。Monad 在去中心化与可扩展性的平衡上显著推进了效率边界。

Monad 在五个主要领域引入了优化，构建了一个以太坊兼容的一级区块链，具备每秒处理 10,000 笔交易（tps）的吞吐量、400 毫秒的区块生成频率以及 800 毫秒的交易确认时间：

- **MonadBFT**：提供高效且抗尾部分叉的拜占庭容错（BFT）共识机制
- **RaptorCast**：实现高效的区块传输
- **异步执行**：通过流水线化共识与执行，增加执行时间预算
- **并行执行与即时编译（JIT Compilation）**：实现高效的交易执行
- **MonadDb**：提供高效的状态访问

Monad 的改进解决了现有瓶颈，同时为应用开发者（完全兼容 EVM 字节码）和用户（兼容以太坊 RPC API）保持无缝兼容性。

根据需要选择详细程度，请访问 **Monad 用户指南** 或 **Monad 开发者指南**。

### 架构

Monad 的设计注重在普通硬件上实现高性能与可扩展性。后续页面将介绍 Monad 的主要架构变化以及用户界面。

首个 Monad 客户端由 Category Labs 开发，从零开始使用 C++ 和 Rust 语言编写。

- **monad-bft**：Category Labs 实现的 Monad 共识客户端
- **monad**：Category Labs 实现的 Monad 执行客户端

两者均在 GPL-3.0 许可下开源。
